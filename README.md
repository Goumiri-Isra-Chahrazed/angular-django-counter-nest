# CounterNest - Angular + Django

## 📦 Project Structure

* frontend → Angular app
* backend → Django API

## 🚀 Setup

### Frontend

```bash
cd frontend
npm install
ng serve
```

### Backend

```bash
cd backend
conda activate myenv
pip install -r requirements.txt
python manage.py runserver
```

## ✅ Features

* Persistent counter feature
* Up / Down / Reset pages
* Backend integration
